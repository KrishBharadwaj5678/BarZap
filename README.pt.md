<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/favicon/favicon.svg" width="35"/> BarZap</h1>

<p align="center">
  O <strong>BarZap</strong> é um gerador de códigos de barras baseado na web que permite criar códigos de barras instantaneamente.
  Basta inserir seu texto ou número, gerar e obter seu código de barras em poucos segundos.
</p>

![BarZapDemo](https://github.com/KrishBharadwaj5678/BarZap/raw/main/BarZapDemo.png)
