<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/favicon/favicon.svg" width="35"/> BarZap</h1>

<p align="center">
  <strong>BarZap</strong> это веб-приложение для мгновенного создания штрихкодов.
  Просто введите текст или число, нажмите кнопку генерации и получите готовый штрихкод за считанные секунды.
</p>

![BarZapDemo](https://github.com/KrishBharadwaj5678/BarZap/raw/main/BarZapDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/assets/readme/features.gif" width="35"/> Возможности

| Возможность | Описание |
| ------------ | -------- |
| ⚡ Мгновенная генерация | Создание штрихкодов одним нажатием |
| 🖊️ Пользовательский ввод | Создавайте уникальные штрихкоды из текста или чисел |
| 🎨 Современный интерфейс | Простой, понятный и удобный дизайн |
| 🌐 Работает в браузере | Не требует установки, доступен онлайн |
| 📱 Адаптивный дизайн | Отлично работает как на компьютерах, так и на мобильных устройствах |
| 🔒 Быстро и бесплатно | Лёгкое, быстрое и бесплатное использование |
