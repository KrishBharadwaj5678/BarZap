<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/favicon/favicon.svg" width="35"/> BarZap</h1>

<p align="center">
  BarZap is a web-based barcode generator that allows users to create barcodes instantly.
  Just enter your text or number, generate, and get your barcode in seconds.
</p>

![BarZapDemo](https://github.com/KrishBharadwaj5678/BarZap/raw/main/BarZapDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/assets/readme/features.gif" width="35"/> Features

| Feature                      | Description                                         |
| ---------------------------- | --------------------------------------------------- |
| ⚡ Instant Barcode           | Generate barcodes quickly with one click            |
| 🖊️ Custom Input             | Enter text or numbers to create unique barcodes     |
| 🎨 Clean UI                  | Simple and user-friendly interface                  |
| 🌐 Web-Based                 | No installation required, works directly in browser |
| 📱 Responsive Design         | Smooth experience across desktop and mobile devices |
| 🔒 Fast & Free               | Lightweight and easy to use anytime                 |
