# 📦 BarZap

**BarZap** is a fast and simple web-based barcode generator that allows users to create barcodes instantly.
Just enter your text or number, click generate, and get your barcode in seconds.

## ✨ Features

| Feature                      | Description                                         |
| ---------------------------- | --------------------------------------------------- |
| ⚡ Instant Barcode           | Generate barcodes quickly with one click            |
| 🖊️ Custom Input             | Enter text or numbers to create unique barcodes     |
| 🎨 Clean UI                  | Simple and user-friendly interface                  |
| 🌐 Web-Based                 | No installation required, works directly in browser |
| 📱 Responsive Design         | Smooth experience across desktop and mobile devices |
| 🔒 Fast & Free               | Lightweight and easy to use anytime                 |

---

## 🛠️ Tech Stack

| Technology               | Purpose                          |
| ------------------------ | -------------------------------- |
| 🌐 **HTML5**             | Structure of the web application |
| 🎨 **CSS3**              | Styling and responsive design    |
| ⚡ **JavaScript**         | Handles barcode generation logic |
| 📊 **JsBarcode**         | Generates barcode SVG instantly  |
