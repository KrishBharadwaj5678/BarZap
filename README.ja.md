<h1 align="center"><img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/favicon/favicon.svg" width="35"/> BarZap</h1>

<p align="center">
  <strong>BarZap</strong> は、バーコードを瞬時に生成できるWebベースのアプリケーションです。<br>
  テキストまたは数字を入力するだけで、数秒以内にバーコードを作成できます。
</p>

![BarZapDemo](https://github.com/KrishBharadwaj5678/BarZap/raw/main/BarZapDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/assets/readme/features.gif" width="35"/> 主な機能

| 機能 | 説明 |
|------|------|
| ⚡ 高速バーコード生成 | ワンクリックでバーコードを即座に生成 |
| 🖊️ カスタム入力 | テキストや数字からオリジナルのバーコードを作成 |
| 🎨 シンプルなUI | わかりやすく使いやすいインターフェース |
| 🌐 Webベース | インストール不要でブラウザから利用可能 |
| 📱 レスポンシブデザイン | PC・スマートフォン・タブレットに対応 |
| 🔒 高速・無料 | 軽量でいつでも無料で利用可能 |

## <img src="https://github.com/KrishBharadwaj5678/BarZap/raw/main/assets/readme/gettingStarted.gif" width="35"/> はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/BarZap.git
```

### 2️⃣ プロジェクトディレクトリへ移動

```bash
cd BarZap
```

### 3️⃣ ブラウザで開く

`index.html` をお好みのWebブラウザで開くだけです。
